1. Introdução¶

Para este projeto, foi sugerido a escolha de uma base de dados (dentre duas) para elaborarmos uma pergunta de nossa escolha. Fiz a opo pela base de dados do Titanic, que contm os dados de todos os tripulantes (sobreviventes e vtimas).

Perguntas escolhidas:

  • Relao entre sexo e sobreviventes
  • Relao entre faixa etria e sobreviventes

10. Conclusão¶

10.1 Limitações da análise:¶

  1. Dados incompletos
  2. Apenas uma amostra foi retirada dos dados
  3. O nmeros de idades que faltam na amostra, podem desviar os resultados

10.2 Outras variáveis¶

  1. Distino entre passageiros e tripulao - No feita esta distino, e essa uma varivel que pode ter uma influncia na anlise proposta
  2. Ordem de alocao nos botes - Isso pode afetar diretamente no resultado do maior nmero de sobreviventes tendar para um sexo especfico

10.3 Suposição teórica¶

Em relação as idades:¶

  1. Pessoas com idades entre 20-29 e 30-39 tiveram a maior quantidade de sobreviventes
  2. As faixas de idades acima dessas, tem uma queda constante

Em relação ao sexo:¶

  1. Aparentemente, as pessoas do sexo feminino tiveram a maior quantidade de sobreviventes
  2. Notamos tambm uma curva de crescimento conforme a idade das mulheres foi aumentando

Possíveis motivos:¶

  1. Uma suposio com relao ao fato de mulheres e mulheres com idades mais avanadas terem o maior nmero de sobreviventes, pode ser devido ao fato de ser dada prioridade para crianas, mulheres e idosos na utilizao dos primeiros botes salva vidas
